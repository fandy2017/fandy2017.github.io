「推绵羊」 Pushing the sheep 游戏设定

|

< 游戏概念 >

传统推箱子游戏界面缺乏美观性、视觉感受单一、用户体验不强，是否可以推陈出新，带大家重返童年。

|

< 故事背景 >

一只来自远古的绵羊和一个来自现代世界的你，同时掉落进了扭曲的时空间隙，你虽然毫发无伤，但绵羊却寸步难行！只有带着绵羊落入时空隧道、突破重重关卡，才能回到原来的世界！

|

< 游戏规则 >

主角拥有30格体力，每移动一次绵羊消耗1格体力，体力耗尽游戏结束每个关卡藏有大栗树，其果实大力丸可以恢复5格体力。

|

关卡设置：

|

序章 介绍故事背景 解锁第一关

|

第一关

需要推1只绵羊到指定地点 通关后解锁第二关

通关最低移动绵羊次数：6；藏有一颗大栗树；需要推一只绵羊到一个传送阵

|

第二关 需要推2只绵羊到指定地点 通关后解锁第三关

通关最低移动绵羊次数：8；藏有一颗大栗树；需要分别推两只绵羊到两个传送阵

|

第三关 需要推3只绵羊到指定地点 开放大栗树和大力丸 通关后解锁第四关

通关最低移动绵羊次数：12；藏有两颗大栗树；需要分别推三只绵羊到三个传送阵

|

第四关 需要推4只绵羊到指定地点 通关后解锁第终章剧情

通关最低移动绵羊次数：16；藏有两颗大栗树；需要分别推四只绵羊到四个传送阵

|

终章

|

通关奖励：

通关后剩余体力可以转化为积分并记录在榜

第二周目可以继承未使用道具（大力丸）

如四关连续使用最低步数通关，可开启隐藏关卡“天哪，这里有一群迷路的绵羊”

|

< 章节剧情 >

# 序章 时空中的间隙 #

（暗场 有一个女人的模糊影子）

- 少年人……

- 谁？我在哪儿？

- 你现在地处时空中的间隙，被一股特殊的力量传送到了这里。

- 时空穿越？

- 是的，这里的时空秩序崩塌了，每下一层都会倒退一千年。

- 一千年？！那我该怎么回去？

- 别急，完成任务你就能回到那个正常的世界了。

（影子消失）

- 等等…！别走！

- 祝你好运……

（场景亮）

# 无人问津的都市 #

- 嘿！你是谁？

- 谁在说话？

- 就在你眼前呢。

- 你是说你是这只绵羊？

- 没错。好吧，这个时空一定不太正常，连羊都会说话了。（我打算不去管它，找找有没有别的路。）

- 嘿！嘿！我动不了了。

- 你可是一只绵羊，怎么可能不会动呢。

- 谁知道呢，兴许我得了羊癫疯。

- 别开玩笑了，你根本不是羊吧，不然怎么会说人话呢，还会得人类的疾病。

- 远古的羊都是通人话的，你难道不知道吗？

- 我是现代人，怎么可能知道。

- 好吧好吧，可悲的现代人，总之你得帮我回去。

（难道，这就是所说的任务？）

- 我可没力气扛着你跋山涉水。

- 只要把我推到传送阵，我们自然会落到时空隧道中去的。记住保留体力，如果力气用完了，我们也就完了。

- 我明白了，可是，你怎么会来到这儿呢？

- 羊总是迷迷糊糊的，你知道的。

（第一关开始）

# 旧城上的蔷薇花 #

- 你瞧，我们成功地迈出了第一步。

- 那事不宜迟，我们继续赶路……

- 嘿！我感觉得到同伴的气息。

- 你是说还有一只羊？

- 没错，我们得一起回去。

- 它……不会也得了羊癫疯吧……

（第二关开始）

# 通往过去的阶梯 #

- 我说你，年轻人！你不能用点劲儿吗？我可感觉得到我们的同伴正越来越多了。

- 你没看到我已经快精疲力尽了吗？

- 噢，看来的确如此。有一件事忘记告诉你了，在这时空缝隙中长着一种神奇的树，吃了它的果子立刻会变得力大无穷，我们俗称它为大栗树，结的果子叫做大力丸。

- 你怎么不早说！

（第三关开始 需要移动三只绵羊）

# 重返绿野仙踪林 #

- 年轻人，可喜可贺，我们快要到达终点了。

（第四关开始 需要移动四只绵羊）

# 终章 #

-哈哈哈，干得好，我回来啦！

-可你还不能动呢。

-没关系，只要回到了这里，一切都会变得好起来的。

-好吧，你是回来了，我该怎么办呢？

-古老的绵羊一族自古掌握了时空穿越的能力，送你回去不过小菜一碟。

-真的？

-当然，不过生活处处有惊喜，我有预感，也许我们会再见的！

（剩余体力转化成积分 1体力=100分 并记录在榜）
