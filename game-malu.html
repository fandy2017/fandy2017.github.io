车载系统 HMI

纸面原型 / Illustrator

策划 / 平面

暂无试玩

「马路杀手通缉令」 MAROAD  MURDER  WANTED

|

< 应用概念 >

道路拥堵不堪，飙车超车泛滥，本本族和碰瓷小子捣蛋，真是够了！

既然城市马路不成马路，更像是各种种族的栖息的江湖，那么何妨将开车作为游戏，整个世界都是一张冒险地图？司机可以随意为其他司机贴上标签，释放压力娱乐大众的同时提供有效信息，提高危险车辆的危险系数，进而帮助其他司机躲过一劫；在马路上进行友好社交，为周围司机提供帮助，打发堵车的无聊时光；我们还提供别具一格的语音系统，让你告别平淡无奇的电子语音提示。

|

< 应用体验 >

缓解 释放马路压力Relieve road irate symptom

娱乐 驾驶过程游戏化Entertain the driving process

社交 与周围人交流Make drivers to be friends/neighbors

|

< 独特卖点 >

1. 行车过程游戏化！堵车再也不枯燥！

2. 寻路提示语音提供幽默度调节机制，实时更新！

3. 从马路天使到飙车傻虫，数百种不同标签任你选！

4. 引入LBS社交系统，遇见今生的女神！

5. 遇到困难即时求助，远亲不如近邻，老司机就在你身边！

6. 工会系统，集结你的车队，比拼公里数吧！

|

|

< 特殊功能 >

1. 可调节的语音提示幽默度系统

例如机械系、高冷系、萌妹系

ex. 前方车子的人昨天被炒鱿鱼了，请不要和他计较。

ex. 当心！你的引擎像是热锅上的蚂蚁。

ex. 目标设定：五角场，预计时间：你再打一个哈欠就到了

ex. 碰瓷王正在靠近，请注意躲避。

|

2. 定制语音系统

基于车内良好的静音环境，应用可以为不同人群录制声音，例如恋人、父母、子女都可以为车辆定制独有的语言指令，让一个人的行车路更加温馨。

|

3. 基于LBS的社交系统

好友系统

呈现方式：弹幕 / 私聊

|

4. 标签系统

以车辆为单位，在初始阶段，所有数值为0，为防止误评，累计3次来自其他车辆所标记的同一标签视作一次有效点数，该标签数值增加1点。数值超过一定点数，该车安全系数减少一格，颜色从绿色向红色逐渐转变。

如果该驾驶员的行车记录长期良好，一次标记可记作2点，同时拥有更多次数的世界频道发言权，而危险等级越高的驾驶员则没有世界频道（弹幕频道）的发言权。
